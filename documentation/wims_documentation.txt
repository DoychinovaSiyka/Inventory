WAREHOUSE INVENTORY MANAGEMENT SYSTEM (WIMS)
Автор: Сийка Дойчинова


1. ОПИСАНИЕ
WIMS е модулна система за управление на складове (ООП, Python 3.x).
Следва MVC архитектура и съхранява данни в JSON формат.

2. СТРУКТУРА НА ПРОЕКТА - Ключови директории
------------------------------------------------------------
/models/      - Дефиниция на бизнес обектите (Product, User, etc.)
/controllers/ - Бизнес логика и CRUD операции
/views/       - CLI интерфейс и Utility (password_utils)
/storage/     - Работа с JSON Repository
/validators/  - Логическа проверка на входните данни
/graph/       - Алгоритмичен модул (Dijkstra за локации)
main.py       - Входна точка на приложението



3. ИНСТРУКЦИИ ЗА СТАРТИРАНЕ
------------------------------------------------------------
1) Стартирайте: python main.py
2) Тестови потребители:
   - Администратор: admin / admin123
   - Оператор: dimitar_01 / dimitar01



4. ТЕХНИЧЕСКИ ОСОБЕНОСТИ
------------------------------------------------------------
- Маскиране на пароли: реализирано в password_utils.py (msvcrt/getpass).
- Разпределена наличност: Проследяване на количества по конкретни локации (W1, W2...).
- Логистика: MOVE операциите актуализират инвентара двустранно без финансова транзакция.
- Сигурност: Декоратор @require_password защитава критичните менюта.

5. ИЗПОЛЗВАНИ БИБЛИОТЕКИ (Standard Library Only)
------------------------------------------------------------
- json, os, sys, pathlib (Данни и Файлове)
- datetime, uuid (Време и ID)
- msvcrt, getpass, functools (Сигурност)
